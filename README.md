# Tim Cao

This is the source code for my personal site: [timcao.com](https://timcao.com).

It is a small Next.js site for projects, experiments, music, photos, and whatever else I want to put on the internet without making it too serious.

## Local Setup

```bash
npm install
npm run dev
```

The local site runs at [http://localhost:3000](http://localhost:3000).

## Checks

```bash
npm run check
```

That runs linting, type checks, tests, and a production build.

The site is deployed with Vercel.
